$$\left( (\forall x) [\text{Being}(x) = \text{Being}(y)] \right) \oplus \left( (\forall x,y) [x \neq y] \right)$$

Formal English: The disjunctive synthesis $\oplus$ asserts both a universal sameness of being and universal distinctness simultaneously ‚Äî unity of being coexisting with pervasive difference.

$$E: (S, \delta) \to (S', \delta) \quad \text{(evolution explores fixed } \delta \text{)}$$

Formal English: Evolution $E$ maps state $S$ with parameter $\delta$ to a new state $S'$ while keeping the landscape parameter $\delta$ fixed ‚Äî evolution operates within an unchanged constraint set.

Formal English: Generativity $G$ maps $(S, \delta)$ to $(S', \delta')$ changing both the state and the landscape/constraints $\delta$ ‚Äî generativity rewrites the transformation rules themselves.
